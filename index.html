Digit Recognition

I wrote matlab and python scripts that can recognize hand-written digits using Machine Learning. I used applications of linear algebra such as Singular Value Decomposition as a machine learning algorithm. This method was also used by US postal office to recognize hand-written digits. I also wrote a paper by Latex.
