Google Compute Engine

I worked directly with research scientist Michael Friedlander as research assistant. I was responsible to set up the cloud-based grid engine structure with master node and worker nodes. Most of the development of this project was done with Python and bash scripting. Our code was written on top of the Google Standard Library for Google Compute Engine. The distributed cloud clustering allows computational optimizations which can also be used in Decentralized Blockchain!

Digit Recognition

I wrote MATLAB and Python code that can recognize hand-written digits using Machine Learning techniques. I used applications of linear algebra such as Singular Value Decomposition (SVD) as the core algorithm. This method was also used by the U.S. postal office to recognize hand-written addresses on mails. My paper below explains the project details and implementations.

File Compression

Fast, efficient and new algorithm using Huffman Tree encoding to manipulate the bits of any files and compress the size. The code base is in C/C++ and I collaborated on this project with a college classmate, Matthew Kotila. Our algorithm is significantly fast but we still would like to improve the performance with better speed, optimization and time complexity.

IEEE 802.11 Wireless LAN

Wrote CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance) Network Protocol with Discrete Event Simulations using negative exponential distribution of random variables. Markov Chain models were used to calculate the probability distributions and M/M/1 queue model for single server side. The client side is the command line where simulations can be customized. There is an additional technical README document on its GitHub page. I collaborated on this repo with Allen Speers
