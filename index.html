Google Compute Engine

I worked directly with research scientist Michael Friedlander as research assistant. I was responsible to set up the cloud based grid engine structure with master node and worker nodes. Most of the development of this project was done with python and bash scripting. Our code was written on top of the Google Standard Library for Google Compute Engine. The distributed cloud clustering allows computational optimizations which can also be used in Decentralized Blockchain!

Digit Recognition

I wrote matlab and python code that can recognize hand-written digits using Machine Learning techniques. I used applications of linear algebra such as Singular Value Decomposition (SVD) as the core algorithm. This method was also used by U.S. postal office to recognize hand-written addresses on mails. IEEE 802.11 Wirless LAN

Wrote CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance) Network Protocol with Discrete Event Simulations using negative expontential distribution of random variables. Markov Chain models were used to calculate the probability distributions and M/M/1 queue model for single server side. The client side is the command line where simulations can be customized. There is an additional technical README document on its Github page. I collaborated on this repo with Allen Speers
