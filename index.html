Google Compute Engine

I worked directly with my professor Michael Friedlander as research assistant. I was responsible to set up the cloud based grid engine structure with master node and worker nodes. Most of the development of this project was done with python and bash scripting. Our code was written on top of the Google Standard Library for Google Compute Engine. The distributed cloud clustering allows computational optimizations which can also be used in Decentralized Blockchain!

Digit Recognition

I wrote matlab and python code that can recognize hand-written digits using Machine Learning. I used applications of linear algebra such as Singular Value Decomposition as core algorithm. This method was also used by US postal office to recognize hand-written digits. Paper is written by me on its details and implementations.

IEEE 802.11 Wirless LAN

Wrote CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance) Network Protocol with Discrete Event Simulations using negative expontential distribution of random variables. Markov Chain models were used to calculate the probability distributions and M/M/1 queue model for single server side. The client side is the command line where simulations can be customized.
